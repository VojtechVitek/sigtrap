trap: Trap and print signals.
======

# Usage
```bash
$ trap
^Cinterrupt
^Cinterrupt
user defined signal 1
user defined signal 2
alarm clock
child exited
trace/BPT trap
continued
quit
interrupt
quit
Killed: 9
```

# Installation

```bash
go get -u github.com/VojtechVitek/trap
```
*You might need to [download Go](https://golang.org/dl/) first.*
